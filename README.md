Indian Car Sales Analysis Dashboard using Power BI
Project Overview

This project presents an interactive Power BI dashboard developed to analyze Indian car sales data and generate meaningful business insights. The dashboard helps understand overall sales performance, company-wise and model-wise trends, segment distribution, and time-based performance patterns.

The main objective of this project is to transform raw sales data into visually informative insights that support data-driven decision-making. It provides a comprehensive overview of key performance indicators and highlights important trends in the automobile market.

Objectives

Analyze overall car sales performance

Identify top-performing companies and models

Understand sales trends across different segments and body types

Evaluate monthly and quarterly sales performance

Measure growth using Month-on-Month (MoM) and Year-on-Year (YoY) analysis

Provide interactive and dynamic data visualization

Key Features

KPI Summary including Total Revenue, Average Revenue, Total Cars Sold, and Model Count

Company-wise Sales Analysis

Model-wise Performance Analysis

Segment-wise and Body Type Distribution Analysis

Monthly Sales Trend Analysis

Quarterly Sales Performance Comparison

MoM and YoY Growth Analysis

Interactive filters for dynamic data exploration

Tools and Technologies Used

Power BI

DAX (Data Analysis Expressions)

Data Visualization Techniques

Dashboard Components
Overview Dashboard

Provides high-level summary including key performance indicators, company-wise sales, model performance, and segment distribution.

Sales Trend Dashboard

Focuses on time-based analysis including monthly and quarterly sales trends, as well as growth analysis using MoM and YoY metrics.

Business Insights Generated

Identified top-performing companies such as Maruti, Hyundai, and Tata

Highlighted best-selling car models

Revealed highest contributing vehicle segments

Analyzed seasonal and quarterly sales patterns

Evaluated sales growth trends over time

This dashboard demonstrates the ability to design professional business intelligence solutions using Power BI. It converts raw data into meaningful insights that help in understanding market performance, identifying trends, and supporting business strategy.
